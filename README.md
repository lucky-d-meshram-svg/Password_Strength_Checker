# Password_Strength_Checker

<img width="412" height="412" alt="password" src="https://github.com/user-attachments/assets/0d6b883f-d59a-4d31-a965-538ce2b9c99a" />

Objective:
To build a Password Strength Checker that evaluates passwords based on security criteria and provides feedback for improvement

1. Checks password length and complexity

2. Validates the use of uppercase and lowercase letters

3. Detects numbers and special characters

4. Provides real-time feedback on password strength

5. Suggests improvements for weak passwords

6. Simple and user-friendly interface
